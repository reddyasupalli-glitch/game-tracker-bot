Title: Privacy Policy – Game Tracker Bot

Effective Date: January 16, 2026

Game Tracker Bot (“we”, “our”, “the bot”) respects your privacy.

1. Data We Collect

The bot collects only the following data to provide game tracking features:

Discord User ID

Discord Username

Server (Guild) ID

Game activity name (example: Valorant, GTA V)

Session start and end timestamps

Play duration

2. How We Use Data

This data is used only for:

Showing your played games history

Showing stats such as playtime and leaderboard

3. Data Sharing

We do not sell or share your data with third parties.

4. Data Storage

Data is stored securely in the bot database.

5. Data Removal

Users can remove their data any time using:
